Favicon assets:
- favicon.svg (single scalable source)
- favicon-32.png (OPTIONAL: generate from svg)
- favicon-192.png (OPTIONAL: generate from svg)
- apple-touch-icon.png (OPTIONAL: 180x180 recommended)

Currently only favicon.svg exists. You can generate PNG variants:
1. Using inkscape (example):
   inkscape favicon.svg -o favicon-32.png -w 32 -h 32
   inkscape favicon.svg -o favicon-192.png -w 192 -h 192
   inkscape favicon.svg -o apple-touch-icon.png -w 180 -h 180

Or use an online generator.
